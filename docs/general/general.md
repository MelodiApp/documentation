# General


Este apartado resume el contexto, los objetivos y la visión global del proyecto *MelodiApp*, una aplicación inspirada en Spotify desarrollada como parte de la materia **Ingeniería del Software 2**.

## Objetivos del Proyecto
El objetivo principal del proyecto fue **diseñar, implementar y desplegar una aplicación de reproducción de música**, con funcionalidades similares a Spotify, que permitiera a los usuarios:

- Buscar y reproducir canciones.
- Crear y gestionar playlists.
- Interactuar con contenido musical de forma intuitiva.
- Autenticarse y mantener una sesión segura.

El proyecto incluyó también el desarrollo de un **backoffice web** para la administración del sistema.

## Descripción del Proyecto
MelodiApp es una aplicación completa que integra distintos componentes tecnológicos y arquitectónicos.  
El desarrollo implicó:

- **Arquitectura basada en microservicios**, desacoplando las responsabilidades en servicios independientes.
- **Despliegue en la nube**, asegurando escalabilidad y disponibilidad.
- **Un frontend móvil**, orientado a la experiencia del usuario final para la reproducción de música.
- **Un frontend web para backoffice**, destinado a la administración de usuarios, catálogos y contenido.
- **Un backend robusto**, encargado de la lógica del negocio, la comunicación entre servicios y la persistencia de datos.

Todo el sistema fue construido respetando principios de diseño como modularidad, mantenibilidad y separación de responsabilidades.
